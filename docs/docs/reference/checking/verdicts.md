# Verdicts

{{ decorated_definition("Verdict") }}

Here are listed all defined verdicts and their descriptions.


**verdict:NC**
:   Not checked.
    
    Solution is waiting for it to be checked 

**verdict:OK**
:   Ok.
    
    Everything's fine, your solution is fully functional.

**verdict:WA**
:   Wrong answer.
    
    Your solution gives a wrong answer at some moment.

**verdict:CE**
:   Compilation error.
    
    Your solution did not compile sucessfully.

**verdict:RE**
:   Runtime error.
    
    Your solution fails at runtime.

**verdict:PE**
:   Presentation error.
    
    Your solution gives answer in a wrong format.

**verdict:PCF**
:   Precompile check failed.
    
    Your solution has failed to match prerequisites and constraints.

**verdict:TL**
:   Time limit.
    
    Your solution has been running for too long.

**verdict:ML**
:   Memory limit.
    
    Your solution has been taking too much memory.

**verdict:TSF**
:   Testing system failed.
    
    Testing system has unexpectedly failed, indicates a bug.
