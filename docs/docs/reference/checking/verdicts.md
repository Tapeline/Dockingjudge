# Verdicts

{{ decorated_definition("Verdict") }}

Here are listed all defined verdicts and their descriptions.

| Code  | Short name              | Description                                                     |
|-------|-------------------------|-----------------------------------------------------------------|
| `NC`  | Not checked             | Solution is waiting for it to be checked                        |
| `OK`  | Ok                      | Everything's fine, your solution is fully functional            |
| `WA`  | Wrong answer            | Your solution gives a wrong answer at some moment               |
| `RE`  | Runtime error           | Your solution fails at runtime                                  |
| `PE`  | Presentation error      | Your solution gives answer in a wrong format                    |
| `PCF` | Precompile check failed | Your solution has failed to match prerequisites and constraints |
| `TL`  | Time limit              | Your solution has been running for too long                     |
| `ML`  | Memory limit            | Your solution has been taking too much memory                   |
| `TSF` | Testing system failed   | Testing system has unexpectedly failed, indicates a bug         |
