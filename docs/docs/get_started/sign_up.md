# Participation

## Register or login

To participate in a contest, you first need to log into your account.
If you don't have one, you can register.

![](../img/ui-register.png)

## Enter contest

Select the contest you want to participate in 

![](../img/ui-contest-list.png)

then head to the start page and press this button.

![](../img/ui-enter-contest.png)

After you've done it, you can solve the tasks.
