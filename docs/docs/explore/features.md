# Features

Dockingjudge provides essentials to host programming contests:

- [x] contest management
- [x] tasks with short answers (regex checking)
- [x] tasks with coding (test-based checking)
    - [x] time and memory limiting
    - [x] precompile checking
- [x] information pages
- [x] markdown on pages
- [x] standings
- [x] viewing of own solutions

## Compilers

Currently, Dockingjudge provides only Python 3.12 as a checking backend.
